## 1. Tópicos e Palavras-Chave de Atuação


* **Desenvolvimento de jogos digitais sérios:** criação de jogos com objetivos educacionais.
* **Game-Based Learning:** aprendizagem baseada no uso estruturado de jogos digitais.
* **Desenvolvimento de sistemas computacionais com gamificação:** aplicação de elementos de jogos para engajamento de usuários de sistemas computacionais.
* **Computação de alto desempenho:** processamento computacional otimizado utilizando CPU e/ou GPU.

## 2. Preferências para Projetos de TCC/Pesquisa

* **Foco Principal:** aplicação de soluções computacionais para problemas reais.
* **Pré-requisitos Mínimos:** O aluno deve ter familiaridade com programação, latex/overleaf e github.

## 3. Disponibilidade e Regras

* **Disponibilidade Atual:** limite de 05 orientações
  - Confirmados (4)
  - Aguardando confirmação (1)
* **Horário de Atendimento:** agendamento prévio via e-mail. Atendimento presencial nas segundas e quintas-feiras.
* **Documentação:** espera-se que o aluno mantenha um repositório Git/GitHub organizado para o projeto.
