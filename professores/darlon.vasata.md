## Computação de Alto Desempenho (HPC)

Projetos voltados à execução paralela e distribuída de aplicações, com foco em desempenho, escalabilidade e uso eficiente de recursos computacionais.

A Computação de Alto Desempenho (*High Performance Computing* — HPC) envolve o uso de múltiplos processadores ou múltiplas máquinas para executar aplicações com alto custo computacional de forma paralela e eficiente. O foco está em paralelismo, escalabilidade e otimização de desempenho.

### Ferramentas principais

- MPI (*Message Passing Interface*): paralelismo distribuído entre nós diferentes (memória distribuída).
- OpenMP: paralelismo em memória compartilhada usando múltiplas *threads*.

### Competências envolvidas

- Sistemas distribuídos
- Linux e linha de comando
- Algoritmos e estruturas de dados

### Pré-requisitos

- Programação em C, C++ ou Python
- Noções de sistemas operacionais
- Conhecimentos básicos de redes de computadores

### Possíveis aplicações

- Simulações científicas e físicas
- Processamento massivo de dados
- Treinamento de modelos de aprendizado de máquina
- Renderização ou processamento de imagens
- Sistemas distribuídos em *clusters* com Slurm



## Grafos

Projetos baseados em modelagem de problemas por meio de grafos e desenvolvimento de algoritmos para análise de relações, caminhos e estruturas complexas.

A área de Grafos estuda estruturas formadas por vértices (nós) e arestas (ligações), utilizadas para representar relações entre entidades. É uma base fundamental em algoritmos, redes e ciência de dados.

### Conceitos principais

- Grafos direcionados e não direcionados
- Caminho mínimo (Dijkstra, A*)
- Árvores, fluxos e conectividade
- Grafos ponderados

### Competências envolvidas

- Lógica e raciocínio algorítmico
- Estruturas de dados
- Programação em Python ou Java

### Pré-requisitos

- Algoritmos e Estruturas de Dados

### Possíveis aplicações

- Análise de redes sociais
- Rotas e logística (GPS, mobilidade urbana)
- Sistemas de recomendação
- Modelagem de redes de computadores
- Análise de dependências em software


## Contêineres e Orquestração

Projetos focados na construção, implantação e escalabilidade de aplicações modernas utilizando contêineres e plataformas de orquestração.

O ecossistema de contêineres e orquestração permite empacotar aplicações com suas dependências e executá-las de forma isolada, escalável e reprodutível.

### Ferramentas principais

- Docker: criação de imagens e contêineres
- Docker Compose: aplicações multi-serviço
- Kubernetes: orquestração, escalabilidade automática, deploy contínuo
- Arquiteturas baseadas em microsserviços

### Competências envolvidas

- DevOps e infraestrutura como código
- Sistemas distribuídos
- Redes de computadores
- Integração contínua (CI/CD)
- Desenvolvimento web

### Pré-requisitos

- Conhecimentos básicos de Linux
- Conceitos de redes e HTTP
- Programação backend/frontend


### Possíveis aplicações

- Plataformas *web* escaláveis (Flask, Svelte, React, etc.)
- Sistemas educacionais distribuídos
- CI/CD e automação de deploy
- Infraestrutura como código
- Monitoramento e observabilidade de serviços
